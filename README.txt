EDIGITAL FAVICON PACKAGE

1. Upload every file in this folder to your website root folder.
2. Copy the contents of head-code.html into the <head> section of every page.
3. Clear the browser cache after replacing an older favicon.

Included:
- favicon.ico: multi-size icon for all major and older browsers
- favicon.svg: high-quality scalable icon for modern browsers
- PNG favicons: 16px through 256px
- apple-touch-icon.png: iPhone and iPad home-screen icon
- Android/PWA icons: regular and maskable versions
- safari-pinned-tab.svg: Safari pinned tab
- mstile-150x150.png and browserconfig.xml: Windows tiles
- site.webmanifest: installable web app icon settings

Brand colour: #40B3FF
